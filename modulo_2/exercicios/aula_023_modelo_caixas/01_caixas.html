Modelo de Caixas

O que são os modelos de caixas?

Todo conteúdo em HTML são demonstrados em uma hierarquia de caixas

Todo elemento visível em HTML é uma caixa. Por exemplo: h1, h2 etc

Composição das caixas

  • height - altura da caixa
  • width - largura da caixa
  • border - borda da caixa
  • padding = espaço dentre a borda e o conteúdo
  • margin - espaço externo da borda
  • outline - contorno a borda da caixa

Tipos de Caixas

  • Box-level = pula para linha de baixo. Exemplos: div, h1, p, main, header, nav, article, aside, footer, form, video
  • Inline-level = pula para o lado, dentro da linha. Exemplos: span, a, code, small, strong, em, sup, label, button, input, select
